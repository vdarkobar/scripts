# scripts
Proxmox scripts

### Quick Install:  
  
```bash
mkdir -p scripts && \
wget -O scripts/npm-podman.sh https://raw.githubusercontent.com/vdarkobar/scripts/main/npm-podman.sh && \
chmod +x scripts/npm-podman.sh && \
./scripts/npm-podman.sh 
```

mkdir -p scripts && \
